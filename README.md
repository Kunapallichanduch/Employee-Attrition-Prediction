# Employee Attrition Prediction using Machine Learning
This project compares multiple machine learning algorithms to predict employee attrition.
## Algorithms Used
- Decision Tree
- Random Forest
- XGBoost
- Naive Bayes

## Objective
Predict whether an employee is likely to leave the organization.

## Comparison Results

| Algorithm | Accuracy |
|----------|----------|
| Decision Tree | 82% |
| Random Forest | 89% |
| XGBoost | 91% |
| Naive Bayes | 79% |

## Technologies
- Python
- Pandas
- Scikit-learn
- XGBoost
